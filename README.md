# Invoice Management App

A modern React-based application for managing invoices, receipts, and related financial documents.

## Features

- Create and manage invoices
- Generate receipts
- Filter and view invoices
- Form validation for data entry
- Responsive design with modern UI components

## Technologies Used

- **React** - Frontend framework
- **React Router** - Client-side routing
- **Vite** - Build tool and development server
- **CSS** - Styling

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd invoice-management
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Usage

1. Start the development server:
   ```bash
   npm run dev
   ```

2. Open your browser and navigate to `http://localhost:5173` (or the port shown in the terminal).

3. Build for production:
   ```bash
   npm run build
   ```

4. Preview the production build:
   ```bash
   npm run preview
   ```

## Project Structure

- `src/components/` - React components organized by feature
- `src/utils/` - Utility functions and data
- `public/` - Static assets
- `src/images/` - Image assets
